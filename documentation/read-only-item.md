# Overview
Use when the resource is specific to get an item.

## Method 
GET

## Required Field
- **displayName**
  - display name to method

- **errorType**
  - [error](link para pagina de erro)

- **itemType**
  - output item type

- **itemExample**
  - output item example (.json/.raml)

## Response
content-type: application/json
HTTP: 200

## Response Error
[response-error](link para pagina de erro)
