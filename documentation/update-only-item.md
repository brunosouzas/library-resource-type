# Overview
Use when the resource is specific to update a item.

## Method 
PUT

## Required Field
- **displayName**
  - display name to method

- **inputType**
  - input data type

- **inputExample**
  - input example (.json/.raml)

- **errorType**
  - [error](link para pagina de erro)

## Response
content-type: application/json
HTTP: 204
